# Shopping Cart JS (Carrito de Compras)

Aplicación web sencilla de **carrito de compras** hecha con **HTML/CSS + JavaScript**, empaquetada con **Webpack** y ejecutada con **webpack-dev-server**.

Permite añadir productos con precio y stock, buscar productos y mostrar el contenido del carrito.

---

## Funcionalidades

- Añadir productos (nombre, precio, stock)
- Listar contenido del carrito
- Buscar producto dentro del carrito
- Validaciones básicas (campos vacíos / datos numéricos)

---

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Node.js + npm
- Webpack + webpack-dev-server

---

## Requisitos

- Tener instalado **Node.js** (incluye npm)

Comprobar versiones:

```bash
node -v
npm -v

## Instalación y ejecución (modo desarrollo)

- Clonar el repositorio:

git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git


- Entrar en la carpeta del proyecto:

cd TU_REPOSITORIO


- Instalar dependencias:

npm install


- Arrancar el servidor de desarrollo:

npm start


- Abrir en el navegador:

http://localhost:8080
